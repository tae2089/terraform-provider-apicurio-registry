# Apicurio Provider

The **Apicurio** provider is used to interact with [Apicurio Registry](https://www.apicur.io/registry/), an open-source API and schema registry. It allows you to manage artifacts (schemas) and their validation/compatibility rules as Terraform resources.

## Authentication

The provider connects directly to the Apicurio Registry REST API. Currently, authentication is handled via the endpoint URL. Future versions may support token-based authentication.

## Example Usage

```terraform
provider "apicurio" {
  endpoint               = "http://localhost:8080/apis/registry/v3"
  keycloak_server_url    = "" #optional
  keycloak_client_id     = "" #optional
  keycloak_client_secret = "" #optional
  keycloak_realm         = "" #optional
}
```
